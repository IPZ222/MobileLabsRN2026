# Лабораторна робота №3: Використання кастомних жестів у React Native та стилізація інтерфейсу мобільного застосунку.

## Мета

Навчитися працювати з жестами користувача у мобільному застосунку, реалізувати взаємодію через різні типи жестів та застосувати сучасні підходи стилізації у React Native.

## Опис реалізованого функціоналу

### 1. Інтерактивні жести

Реалізовано комплексний ігровий елемент ABOBA, який реагує на широкий спектр дій завдяки сучасній системі GestureDetector:
- **Taps & Long Press:** Одинарний тап (1 очко), подвійний тап (10 очок) та утримання протягом 800 мс (50 очок) з використанням Gesture.Exclusive.
- **Pan Gesture:** Можливість вільного переміщення об'єкта по всьому екрану зі збереженням нової позиції після відпускання.
- **Fling:** Швидкі свайпи вліво та вправо для нарахування випадкової кількості бонусних балів (від 1 до 100).
- **Pinch:** Збільшення масштабу об'єкта двома пальцями з плавним поверненням до початкового розміру за допомогою пружинної анімації (withSpring).
- **Оптимізація:** Використано функцію `runOnJS` з react-native-reanimated для безпечної та стабільної синхронізації між UI-потоком жестів та JS-потоком стану React.

### 2. Глобальний стан

Для керування даними між незалежними екранами навігації впроваджено власне централізоване сховище `GameContext`:
- **Quest Tracking**: Відстеження статистики різних дій (кліки, свайпи, щипки) для автоматичного заповнення прогрес-барів у 6 завданнях.
- **Live Updates**: Миттєве оновлення загального рахунку та статусу завдань на всіх екранах без необхідності ручного оновлення чи передачі через пропси.
- **Глобальні налаштування**: Зберігання стану вібрації та конфігурації гри з можливістю повного скидання прогресу через `Alert`.

### 3. Динамічна стилізація

Додаток підтримує повноцінну світлу та темну теми:
- **Dynamic Styles:** Комбінація статичного StyleSheet та динамічних інлайн-стилів, які миттєво реагують на зміну стейту в GameContext.
- **Адаптивний Drawer:** Навігація через `@react-navigation/drawer` повністю синхронізована з темою.
- **Інтерфейс:** Всі UI-елементи перемикаються між світлою та темною палітрами при використанні Switch у налаштуваннях.

## Висновки

У ході роботи опановано створення інтерактивних інтерфейсів у React Native. За допомогою react-native-gesture-handler (GestureDetector) та reanimated реалізовано складні жести: тапи, свайпи (Fling), перетягування (Pan) та масштабування (Pinch).
Використання Context API забезпечило ефективне керування глобальним станом (очки, статистика, налаштування), а динамічна стилізація дозволила впровадити адаптивну темну тему. Застосування runOnJS гарантувало стабільну роботу додатка при взаємодії між UI та JS потоками. Застосунок є архітектурно правильним та оптимізованим.
